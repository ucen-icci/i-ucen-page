Ecuaciones segundo grado

Una ecuación de segundo grado tiene dos soluciones, y tiene la forma ax² + bx + c = 0

Ejercicios:

  1. \[2x^2+ 3x – 2 = 0\]
  2. \[2x^2– 5x = 0\]
  3. \[9x^2– 18x + 8 = 0\]
  4. \[x^2– 3x – 10 = 0\]
  5. \[x^2– 3x = 0\]
  6. \[2x^2– x – 6 = 0\]
  7. \[(x – 3)(x – 1) = 15\]
  8. \[3x^2= 4x\]
Soluciones:
  1. x1 = -2 | x2 = 1/2
  2. x1 = 0 | x2 = 5/2
  3. x1 = 2/3 | x2 = 4/3
  4. x1 = -2 | x2 = 5
  5. x1 = 0 | x2 = 3
  6. x1 = -3/2 | x2 = 2
  7. x1 = -2 | x2 = 6
  8. x1 = 0 | x2 = 4/3
