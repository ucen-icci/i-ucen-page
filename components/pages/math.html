Ecuaciones primer grado.
Ejercicios
Para resolver:
  1. Eliminamos denominadores calculando el mínimo común múltiplo.
  2. Calculamos cuánto le falta a cada denominador para llegar a su mínimo como múltiplo.
  3. Eliminamos los denominadores.
  4. Agrupamos términos x en un lado de la igualdad.
  5. Resolvemos.
  • Ejemplo:
    1. \[\frac{2x-5}{3}+ \frac{3x-6}{4} = \frac{4x-8}{12}\]
      2. mínimo común múltiplo: 12
    2. \[4(2x-5) + 3(3x-6) = 1(4x-8)\]
    3. \[10x - 20 +9x - 18 = 4x - 8\]
      4. Determinamos la igualdad, pasando a un lado nuestras incógnitas, y a un lado nuestros valores numéricos.
    4. \[ 10x + 9x - 4x = -~8 + 20 + 18\]
    5. \[15x = 30\]
    6. \[ x = \normalsize\frac{30}{15}\scriptsize => x = 2\]
      7. El valor de x = 2
Ecuaciones segundo grado.
Ejercicios Video
Una ecuación de segundo grado tiene dos soluciones, y tiene la forma ax² + bx + c = 0

Para resolver:

  1. Se designa los tipos de soluciones que obtendremos, donde discriminante se determina como:
    1. b² - 4ac > 0 | Dos soluciones distintas.
    2. b² - 4ac = 0 | Dos soluciones iguales.
    3. b² - 4ac < 0 | No soluciones.
  2. Determinamos nuestras soluciones utilizando fórmula Bhaskara:
    3. Fórmula Bhaskara \[{x =}\frac{-b~\pm~\sqrt{b^2 - 4ac}}{2a}\]
  • Ejemplo:
    1. \[3x^2 - 6x + 3 = 0\]
      2. Aplicamos discriminante
    2. \[(-6)^2 - 4\times3\times3 => 36 - 36 = 0\]
      3. Discriminante = 0, por lo tanto habrán dos soluciones iguales.
    3. \[\frac{6 \pm \sqrt{36 - 36}}{6}= x\]
      4. x1 = 1
      x2 = 1
Ecuaciones racionales
Ejercicios Video
Son aquellas las que la incógnita aparece en fracciones algebraicas; buscando la igualación de denominadores para tener una ecuación básica.

Para resolver:

  1. Eliminamos los denominadores calculando y multiplicando por el mínimo común múltiplo.
  2. Comprobación de solución. Donde nunca será solución aquellos valores que anulen el denominador.
  • Ejemplo:
    1. \[\frac{2}{x+2}-\frac{1}{x-3} = \frac{3}{x^2-x+6}\]
      2. Resolvemos cuadráticas vistas
    2. \[x^2 -x+ 6 = (x-3)(x+2)\]
      3. Despejamos soluciones que no son posibles
      1. \[x - 3 \therefore x = 3\]
      2. \[x + 2 \therefore x = -2\]
      Minimo común múltiplo: (x-3)(x+2)
    3. \[\frac{2(x-3)}{x+2}-\frac{1(x+2)}{x-3} = \frac{3}{x^2-x+6}\]
    4. \[\frac{2x -6 -x - 2}{(x+2)(x-3)}= \frac{3}{(x+2)(x-3)}\]
      5. Al igualar denominadores, podemos ignorarlos.
    5. \[x - 8 = 3 \therefore x = 3 + 8 \therefore x = 11\]
      6. Valor de x = 11
Ecuaciones irracionales
Ejercicios Video
Son aquellas donde la incógnita aparece bajo signos radicales.

Para resolver:

  1. Aislamos una de las raíces.
  2. Elevamos ambos lados de la ecuación a orden del índice de nuestra raíz.
  3. Hay que comprobar si las soluciones son correctas.
  • Ejemplo:
    1. \[\sqrt{2x-1} + \sqrt{x+4} = 6\]
      2. Aislamos una de las raíces.
    2. \[\sqrt{2x-1} = 6 - \sqrt{x+4}\]
      3. Elevamos ambos lados de la ecuación al índice: 2
    3. \[(\sqrt{2x-1})^2 = (6-\sqrt{x+4})^2 \therefore 2x-1 = 36 + (x+4) - 12\sqrt{x+4}\]
      4. Aislamos la raíz que tenemos nuevamente.
    4. \[12\sqrt{x+4} = 36 + (x+4) - (2x-1) \therefore 12\sqrt{x+4} = 41 - x\]
      5. Elevamos ambos lados de la ecuación al índice: 2
    5. \[(12\sqrt{x+4})^2 = (41-x)^2 \therefore 144(x+4) = 1681 + x ^2 -82x\therefore 144x + 576 = x^2-82x +1681 \]
      6. Resolvemos la ecuación de segundo grado.
    6. \[x^2 - 226x + 1105 = 0 \therefore x^2 - 226x + 1105 = (x-5) (x - 221)\]
      1. \[x-5 \therefore x = 3\]
      2. \[x - 221 \therefore x = 221\]
      Comprobamos ambas soluciones, buscando así la correcta.
    7. \[x1 = 5 => \sqrt{2\times5-1} + \sqrt{5+4} = 6 \therefore \sqrt{9} + \sqrt{9} = 6 \therefore 3 + 3 = 6\]
    8. \[x2 = 221 => \sqrt{2\times221-1} + \sqrt{221+4} = 6 \therefore \sqrt{441} + \sqrt{225} = 6 \therefore 21 + 15 = 6\]
      9. Solución x1 = correcta
      Solución x2 = incorrecta
